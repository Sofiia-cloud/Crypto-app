# Crypto App

Приложение для отслеживания криптовалютных активов. Позволяет просматривать актуальные курсы, добавлять монеты в портфель и рассчитывать их общую стоимость.

## Основные возможности

- Просмотр списка криптовалют с актуальными ценами
- Добавление активов в личный портфель
- Автоматический расчёт общей стоимости инвестиций
- Визуализация динамики изменения цены

## Технологии

### Фронтенд
- **React** — библиотека для построения интерфейсов
- **Ant Design** — компоненты для стилизации
- **JavaScript (ES6+)**

### Бэкенд
- **Node.js** — среда выполнения
- **Express** — веб-фреймворк

## Установка и запуск

### 1. Клонируйте репозиторий
git clone https://github.com/Sofiia-cloud/Crypto-app.git
cd Crypto-app

### 2. Установите зависимости для фронтенда
cd frontend
npm install

### 3. Установите зависимости для бэкенда
cd ../
npm install

### 4. Запустите бэкенд-сервер
node server.js

### 5. Запустите фронтенд
cd frontend
npm run dev
